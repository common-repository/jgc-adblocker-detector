== Description ==

This plugin allows you to display a notice when an ad blocker is detected. This notice can be displayed in a modal box or as text containers in the places where your ads usually appear.

The plugin gives you the option to create an event in Google Analytics to track the use of ad blockers.

== Installation ==

For an automatic installation through WordPress:

1. Go to Plugins > Add New
2. Search for 'JGC AdBlocker Detector'
3. Click 'Install Now' and activate the plugin
4. Go to Settings > JGC AdBlocker Detector

For a manual installation via FTP:

1. Unzip the file 'jgc-adblocker-detector.zip'
2. Upload the 'jgc-adblocker-detector' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings > JGC AdBlocker Detector

To upload the plugin through WordPress, instead of FTP:

1. Go to Plugins > Add New
2. Click 'Upload Plugin' button
3. Select 'jgc-adblocker-detector.zip' file
4. Click 'Install Now' and activate the plugin
5. Go to Settings > JGC AdBlocker Detector
